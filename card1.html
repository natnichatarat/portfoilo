Mobile Application Chatbot

Developed a mobile application using Flutter for a dormitory recommendation chatbot, implementing Dialogflow for user interactions, and utilizing Firebase for database management and user authentication.

Key Features:

  • Users can use the chatbot system
  • owners can add dormitory information and other details of the dormitory
  • Users and owners must register and log in to their accounts
